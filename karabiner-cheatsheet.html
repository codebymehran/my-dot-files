How This Is Organised
Hyper (CapsLock hold)
The base layer. Navigation (hjkl), deletion (m/n), symbols, and global actions. Most frequently used shortcuts live here.
Cmd + Hyper
The extended layer. Selection, secondary actions, multi-char sequences like => and !==, and duplicate tab.
Option + Hyper
The coding layer. Template literals, JSX, console.log/error, arrow functions, ===, !==, ??, spread, optional chaining, and line-level deletion.
Right Option alone
App launcher and system controls. Hold-only — no tap behaviour. Unmapped keys pass through transparently. Reserve for launching and system only.
App-specific (Chrome / VSCode)
Hyper shortcuts that only fire in a specific app via bundle_identifiers condition. Override global keys for context-aware behaviour.
Adding new shortcuts
System actions → Right Option · Frequent actions → Hyper · Secondary → Cmd+Hyper · Coding sequences → Option+Hyper · App-specific → condition block
