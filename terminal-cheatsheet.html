🔧 Optional Tools

brew install eza bat zoxide fnm fzf thefuck
eza Better ls — file icons, colour coding, git status in listings, tree view
brew install eza · replaces ls/ll/la/lt automatically when installed
bat Better cat — syntax highlighting, line numbers, git diff markers in output
brew install bat · replaces cat automatically · use catp for plain output · theme: tokyonight_night
zoxide Smart cd — tracks visited dirs, jump anywhere with a partial name · e.g. z back jumps to backend-labs
brew install zoxide · replaces cd automatically · the more you use it, the smarter it gets
fnm Fast Node version manager — install, switch, auto-switch per project via .nvmrc or .node-version
brew install fnm · usage: fnm install 20 · fnm use 20 · fnm list · auto-switches on cd
fzf Fuzzy finder — Ctrl+R for history search · Ctrl+T for file picker · used by proj command
brew install fzf · preview window shows file contents via bat · Alt+C for fuzzy dir jump
thefuck / fk Mistype a command? Run fk and it auto-corrects it — e.g. git psuh → suggests git push
brew install thefuck · type fk after any failed command to fix it

Setup Notes

Install order Homebrew → Oh My Zsh → zsh plugins → Starship → optional tools → drop config files in
Nerd Font WezTerm uses MesloLGS Nerd Font Mono — install from nerdfonts.com or Starship symbols show as broken squares
zoxide warm-up z doesn't work instantly — use cd normally for a few days and zoxide learns your dirs in the background
fnm vs nvm If you used nvm before, remove it from ~/.zshrc first — fnm and nvm conflict with each other
Plugin errors If shell throws errors on startup, zsh plugins aren't installed yet — install them before copying zshrc
